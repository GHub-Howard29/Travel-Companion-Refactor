# Travel Companion（旅行管理助手）

## 專案目的

Travel Companion 是一套以「旅程（Trip）」為核心設計的旅行管理工具。

最初以旅遊團隊共同記帳為出發點，隨著功能逐步增加，目前已發展為整合旅行記帳、附件管理、離線同步及雲端共享的旅行管理系統。

目前主要提供：

- 共用旅遊帳本
- 個人帳本
- 離線記帳（Offline First）
- 照片附件管理
- 自動同步
- Excel 匯出
- 多幣別
- GitHub Pages 部署

未來將持續加入：

- 旅行資訊（Reference）
- 私人物品準備清單（Checklist）
- 權限管理
- 旅行總覽（Trip Dashboard）

希望所有與旅行有關的資料，都能集中管理於同一套系統內。

---

## 專案理念

本專案希望做到：

1. 即使沒有網路，也能正常使用主要功能。
2. 網路恢復後，自動同步所有待同步資料。
3. 照片與帳目分開同步，降低同步失敗率。
4. 所有資料皆可安全保存於本機。
5. 不需要安裝 APP，只要使用瀏覽器即可操作。
6. 每新增一項功能，都不破壞既有系統架構。
7. 讓未來的自己也能快速理解並維護此專案。

## 開發原則

本專案遵循以下原則：

- 一次只完成一個功能（Feature）。
- 每完成一個功能皆須先完成測試再合併。
- 先討論設計，再修改程式。
- 保持程式模組化，避免再次出現大型 App.tsx。
- 文件採集中整理原則；平常小步實作不逐次更新，於要求整理文件、整理交接文件、準備 Commit 或要求 Commit 時再一次更新。

---

## 專案目標

Travel Companion 的目標，不只是記錄每一筆旅費。

而是希望成為一套能陪伴每一次旅行的管理工具。

希望未來每一次旅行，都只需要開啟 Travel Companion，即可完成旅行前準備、旅行中的記錄，以及旅行結束後的資料保存與回顧。
