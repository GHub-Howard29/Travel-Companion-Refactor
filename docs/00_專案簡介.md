# Travel Companion（旅行記帳助手）

## 專案目的

Travel Companion 是一套以旅遊團隊共同記帳為核心的 Web App。

主要提供：

- 共用旅遊帳本
- 個人帳本
- 離線記帳
- 照片附件
- 自動同步
- Excel 匯出
- 多幣別
- GitHub Pages 部署

---

## 專案理念

本專案希望做到：

1. 即使沒有網路也能完整使用。
2. 網路恢復後自動同步資料。
3. 照片與帳目分開同步，降低失敗率。
4. 所有資料皆可安全保存於本機。
5. 不需要安裝 APP，只要瀏覽器即可使用。
