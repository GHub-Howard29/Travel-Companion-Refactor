# V3.3.0 外幣換算發布交接

更新日期：2026-07-20

## 發布狀態

V3.3.0 已完成 Product Owner 全數回歸驗證與程式提交，待 Product Owner 手動發布／部署。

## 已完成內容

- 外幣換算支援 JPY、KRW、USD、EUR；可記錄換匯、以加權平均匯率估算，並載入臺銀參考牌價供離線使用。
- 計算機同時顯示實際換匯紀錄與臺銀公告牌價兩種新臺幣估算；金額輸入支援千分位。
- 權限：未登入／已登入 Gmail 者僅保留本機資料；管理員／已登記完成的旅程成員，可同步所屬旅程的雲端換匯歷史紀錄。
- 雲端紀錄使用 RLS、Realtime、視窗／計算機聚焦補抓及寫入後回讀；雲端初始化後為唯一權威來源，避免舊快取復活已刪除資料。
- Supabase 正式專案 `travel-companion-db` 已部署 `taiwan-bank-exchange-rate` Edge Function；`exchange_purchases` 已加入 `supabase_realtime` publication。
- 已通過：換匯 CRUD、不同 Trip 隔離、guest/user 本機隔離、編輯者／管理員雲端同步、跨瀏覽器即時更新、刪除不復活、參考匯率載入／離線及雙估算驗證。

## 後續

V3.4.0 前端載入效能優化維持待辦，須待 Product Owner 手動發布 V3.3.0 後才能開始。
