# V3.x

## V3.0.0

開始：

2026/07/06

狀態：

🚧 開發中

版本目標：

建立旅行前準備功能。

已完成內容：

- 旅行資訊（Reference）
- 其他資訊 / 參考資訊本機管理第一階段
- 共同檢查清單本機進度持久化
- 私人確認清單本機版
- 私人確認清單最小雲端同步
- 權限矩陣第一階段落地

私人確認清單目前狀態：

- 入口位於共同檢查清單下方。
- Guest 不顯示入口。
- 登入使用者可新增、編輯、刪除、勾選自己的私人項目。
- 本機資料依 `userEmail + tripId` 保存於 localStorage。
- `trip_editor` / `super_admin` 可同步自己的私人項目到 Supabase。
- `super_admin` / `trip_editor` 不可讀取其他使用者私人清單。
- 尚未做 Pending Queue、Conflict Resolution。

共同檢查清單目前狀態：

- 仍使用 Trip JSON seed + localStorage 勾選進度。
- 下一步接 Supabase shared checklist 最小同步。

# 版本命名規則

例如：

# Git Branch 建議

main

正式版本。

---

develop

開發版本。

---

feature/*

單一功能開發。

例如：

feature/reference

feature/checklist

feature/permission

---

hotfix/*

正式版本 Bug 修正。
