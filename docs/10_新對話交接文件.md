# 一、專案簡介

Travel Companion 是一套以 **Trip（旅行）** 為核心的旅行管理平台。

本次開發目標為 Travel Companion V3。

目前正進行 V3 Foundation 重構。

---

# 二、目前開發階段

目前階段：

> V3 Foundation

目標：

建立所有 Travel Tool 共用的基礎能力。

不是新增功能。

不是修改 UI。

而是建立長期可維護的產品架構。

# 四、產品核心

Travel Companion 採用：

```text
Travel Companion
        │
        ▼
      Trip
        │
        ▼
 Travel Tools
```

Trip 為產品核心。

所有功能皆依附於 Trip。

---

# 五、Foundation

Foundation 負責：

- Current Trip
- Current User
- Workspace
- Permission
- Role
- Sync State

Foundation 不包含任何商業邏輯。

---

# 十三、重要提醒

本專案最重要的架構原則：

Trip 是產品核心。

Foundation 是共同能力。

Travel Tool 是功能模組。

所有新增功能，

皆應新增新的 Travel Tool。

避免修改既有 Tool。

任何架構設計，

皆須符合：

- 易閱讀
- 易維護
- 易擴充
- 不過度設計
- 長期演進
